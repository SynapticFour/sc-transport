# Security Contacts and Response Targets

Use this file for operational security contact paths and response expectations.

## Contacts

- Primary security contact: `contact@synapticfour.com`
- General contact: `contact@synapticfour.com`

## Response targets

- Initial acknowledgement: within 2 business days
- Triage decision: within 5 business days
- Coordinated remediation/disclosure timeline: agreed during triage

## Disclosure policy

- Do not open public issues for vulnerabilities.
- Prefer private disclosure via email or GitHub Security Advisories.
- Credit reporters in advisories unless anonymity is requested.
